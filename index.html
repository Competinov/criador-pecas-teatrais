Como Usar o Sistema

1. Preencha os Módulos

Responda às perguntas de cada módulo com calma. Não há respostas certas ou erradas.

2. Navegue Livremente

Use os botões para avançar ou voltar entre módulos. Suas respostas são salvas automaticamente.

3. Complete no Seu Tempo

Você pode fechar o navegador e voltar depois. Suas respostas estarão salvas.

4. Gere sua Peça

Após preencher todos os módulos, clique em "Finalizar" para gerar sua peça completa.

Dicas Importantes:

  • Seja específico nas suas respostas
  • Use os hints (dicas) como orientação
  • Não se preocupe com perfeição - você pode editar o documento final
  • Se tiver problemas, tente usar o Google Chrome ou Firefox

Sobre o Sistema

Este Sistema Profissional de Dramaturgia Teatral foi desenvolvido por Alexandra Moreira, atriz e dramaturga, com o objetivo de democratizar o acesso a ferramentas profissionais de criação teatral.

O que este sistema faz:

  • Guia você através de 8 módulos essenciais da criação teatral
  • Organiza suas ideias de forma profissional
  • Gera um documento Word completo com sua peça
  • Preserva sua privacidade - tudo fica no seu computador

Para quem é:

  • Dramaturgos iniciantes e experientes
  • Professores de teatro
  • Estudantes de artes cênicas
  • Qualquer pessoa com uma história para contar
