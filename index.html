Como Usar o Sistema

Instruções:

  1. Preencha todos os campos obrigatórios (marcados com *)
  2. Navegue entre os módulos usando os botões
  3. Suas respostas são salvas automaticamente
  4. Ao completar, clique em "Finalizar e Gerar Peça"
  5. Escolha o formato de download (Word ou Texto)

Problemas com Download?

Se o Word não funcionar, use a opção de Texto (.txt) que é mais compatível.

Você também pode visualizar o texto completo e copiar manualmente.
