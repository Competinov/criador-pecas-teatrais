Como Usar o Sistema

Instruções:

  1. Preencha todos os campos obrigatórios (marcados com *)
  2. Navegue entre os módulos usando os botões
  3. Suas respostas são salvas automaticamente
  4. Ao completar, clique em "Finalizar e Gerar Peça"
  5. Escolha o formato de download:
    • Texto (.txt) - Mais compatível, abre em qualquer editor
    • RTF - Mantém formatação, abre no Word
    • HTML - Para visualizar no navegador
    • Copiar - Para colar em qualquer programa

Problemas com Download?

Se tiver problemas, use a opção "Visualizar e Copiar Texto" para copiar o conteúdo manualmente.
