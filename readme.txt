1. O QUE É ESTE SISTEMA
================================================================================

O Sistema Profissional de Dramaturgia Teatral é uma ferramenta digital inovadora
que guia escritores, dramaturgos e criadores teatrais através de um processo
estruturado de criação de peças teatrais completas.

Através de 8 módulos especializados com perguntas orientadoras, o sistema ajuda
a desenvolver todos os aspectos de uma produção teatral - desde o conceito
inicial até os detalhes técnicos de produção.

Ao final do processo, o sistema gera automaticamente um documento Word profissional
contendo toda a documentação da peça, pronta para ser utilizada em produções,
apresentações para teatros, submissões a editais ou registro de direitos autorais.

3. COMO FUNCIONA
================================================================================

PASSO 1: ACESSO
- Abra o sistema em qualquer navegador moderno
- Não requer instalação ou cadastro
- Funciona online através do link fornecido

PASSO 2: PREENCHIMENTO
- Responda às perguntas de cada módulo
- Navegue livremente entre módulos
- Suas respostas são salvas automaticamente

PASSO 3: DESENVOLVIMENTO
- Complete os 8 módulos no seu próprio ritmo
- Pode pausar e retomar quando quiser
- Revise e edite suas respostas conforme necessário

PASSO 4: FINALIZAÇÃO
- Após completar todos os campos obrigatórios
- Clique em "Finalizar e Gerar Peça"
- Baixe seu documento Word completo

6. INSTRUÇÕES DE USO
================================================================================

INICIANDO:
1. Acesse o sistema através do link fornecido
2. Leia os avisos importantes na página inicial
3. Comece pelo Módulo 1: Essência e Propósito

NAVEGANDO:
• Use os botões "Anterior" e "Próximo" para navegar
• Clique nos números dos módulos para pular diretamente
• A barra de progresso mostra o percentual completo

PREENCHENDO:
• Campos com asterisco (*) são obrigatórios
• Leia os hints (dicas) amarelos para orientação
• Tome seu tempo - não há limite de tempo
• Revise e edite conforme necessário

SALVANDO:
• O salvamento é automático a cada mudança
• O botão flutuante confirma o salvamento
• Para backup manual, use "Salvar Progresso"

FINALIZANDO:
• Complete todos os campos obrigatórios
• O botão "Finalizar" aparecerá quando pronto
• Aguarde o processamento (alguns segundos)
• O download do Word iniciará automaticamente

8. PERGUNTAS FREQUENTES
================================================================================

P: Preciso preencher tudo de uma vez?
R: Não. O sistema salva automaticamente e você pode retomar quando quiser.

P: Posso editar minhas respostas depois?
R: Sim. Você pode voltar e modificar qualquer resposta antes de gerar o documento.

P: O que acontece se eu fechar o navegador?
R: Suas respostas ficam salvas. Ao retornar, tudo estará como deixou.

P: Outras pessoas podem ver minhas respostas?
R: Não. Cada usuário tem sua sessão privada e isolada.

P: Posso criar várias peças?
R: Sim. Use "Começar Novo Projeto" para iniciar uma nova peça.

P: O documento gerado é editável?
R: Sim. O arquivo Word pode ser editado normalmente após o download.

P: Funciona em celular?
R: Funciona, mas a experiência é otimizada para telas maiores.

P: Preciso pagar algo?
R: Não. O sistema é gratuito para uso educacional e artístico.

9. RESOLUÇÃO DE PROBLEMAS
================================================================================

PROBLEMA: O documento não é gerado
SOLUÇÃO: 
- Verifique se preencheu todos os campos obrigatórios (*)
- Tente usar o Google Chrome ou Firefox
- Certifique-se que JavaScript está habilitado
- Aguarde o processamento completo (até 30 segundos)

PROBLEMA: Perdi minhas respostas
SOLUÇÃO:
- Verifique se está no mesmo navegador
- Não use modo incógnito/privado
- Evite limpar dados do navegador
- Use sempre o mesmo dispositivo

PROBLEMA: O sistema está lento
SOLUÇÃO:
- Verifique sua conexão com internet
- Feche outras abas do navegador
- Reinicie o navegador
- Tente em horário com menos tráfego

PROBLEMA: Botões não funcionam
SOLUÇÃO:
- Atualize a página (F5)
- Verifique se JavaScript está ativo
- Desative extensões do navegador
- Tente outro navegador

PROBLEMA: Download não inicia
SOLUÇÃO:
- Verifique bloqueador de pop-ups
- Permita downloads do site
- Verifique pasta de downloads
- Tente "Salvar como" se aparecer
